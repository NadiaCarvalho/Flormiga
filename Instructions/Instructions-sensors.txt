1. Blinka setup:

cd ~
sudo pip3 install --upgrade adafruit-python-shell
wget https://raw.githubusercontent.com/adafruit/Raspberry-Pi-Installer-Scripts/master/raspi-blinka.py
sudo python3 raspi-blinka.py

(Reboot when it asks)

pip3 install --upgrade adafruit-circuitpython-dotstar adafruit-circuitpython-motor adafruit-circuitpython-bmp280

2. Voicecard software:

check if BrainCraft HAT is being detected with sudo i2cdetect -y 1 (it should have a 1a in row 10, column a)

cd ~
sudo apt-get install -y git
git clone https://github.com/HinTak/seeed-voicecard
cd seeed-voicecard

check your kernel version using "uname -r"

if it is about ~5.4, use:
git checkout v5.5
sudo ./install.sh

else, if it is equal or higher than 5.10, use:
git checkout v5.9
sudo ./install.sh

(Reboot) - On reboot run "sudo aplay -l" to list all sound cards (you should see one card with seeed2micvoicec [seeed-2mic-voicecard])

After you can test that both, speaker, headphones and micro work with:

speaker-test -c0 in which 0 stands for the number of the card as seen in the last command
sudo arecord -f cd -Dhw:0 | aplay -Dhw:0 in which 0 stands for the number of the card as seen in the last command

You can use alsamixer to adjust the volume, but dont forget to select the card with F6 (or S)

create a configuration so that the bonnet is the default sound card:

sudo nano /etc/asound.conf

add the following lines (in which 0 stands for the number of the card):
defaults.pcm.card 0
defaults.ctl.card 0
